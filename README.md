# 企业级员工办公设备全生命周期自动化管理系统

## 系统概述

本系统是一套完整的企业级设备全生命周期管理解决方案，覆盖设备从申请、采购、入库、领用、维修、盘点到报废的全过程自动化管理。

## 核心功能模块

### 1. 设备申请与采购管理
- **岗位配置校验**: 自动校验员工岗位是否允许申请对应类型设备，检查是否达到配置上限
- **部门预算校验**: 自动校验部门预算额度，超预算拒绝申请
- **智能库存检查**: 自动检查库存，有库存直接分配，无库存自动发起采购
- **供应商询价**: 自动向优选供应商发起询价，获取报价和货期
- **智能选商**: 基于历史评分、报价、货期综合计算，选择最优供应商
- **采购订单生成**: 自动生成采购订单，超50万自动触发总监级审批

### 2. 设备入库与领用管理
- **自动入库**: 采购到货后自动生成设备档案，分配唯一资产编号和二维码
- **自动更新库存**: 入库后自动更新库存数量
- **领用协议签署**: 分配设备时自动生成电子领用协议，员工签署后锁定设备到人
- **设备归还**: 支持设备归还流程，归还后自动更新库存

### 3. 设备报修与维修管理
- **保修状态识别**: 自动识别设备是否在保修期内
- **保内维修**: 在保内自动生成返厂维修单，跟踪物流信息
- **保外维修**: 保外自动估算维修费用，超过阈值触发审批流程
- **工程师指派**: 审批通过后自动指派工程师
- **折旧更新**: 维修完成后自动更新设备档案并重新计算折旧

### 4. 年度盘点管理
- **自动生成盘点任务**: 年度自动生成盘点任务，包含所有设备
- **扫码确认**: 员工扫码确认设备状态（正常/丢失/损坏/不符）
- **差异处理**: 盘点差异自动生成调整记录，明确责任人
- **进度跟踪**: 实时查看盘点进度和统计数据

### 5. 设备报废管理
- **残值校验**: 自动校验设备剩余价值和折旧率
- **报废审批**: 提交报废申请后自动触发审批流程
- **配件回收**: 审批通过后清理数据，记录回收配件信息
- **折旧结算**: 报废时计提最后一期折旧，设备价值清零

### 6. 统计报表与数据导出
- **日报自动生成**: 每天凌晨2点自动统计生成报表
- **使用率统计**: 统计各类型设备使用率
- **故障率统计**: 统计设备故障率和平均维修成本
- **折旧统计**: 统计累计折旧和月度折旧额
- **趋势图表**: 生成近12个月设备管理趋势图
- **PDF/Excel报告**: 自动生成美观的PDF和Excel报告
- **组合查询**: 支持按员工、设备型号、时间范围组合查询
- **批量导出**: 支持全生命周期记录批量导出Excel

### 7. 日志管理与异常预警
- **操作日志**: 记录所有操作日志，支持追溯
- **实时预警**: 异常情况实时推送到企业微信群
- **多级预警**: 支持info、warning、error、critical四级预警

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行系统

```bash
python main.py
```

### 3. 系统菜单

```
1. 初始化数据库
2. 运行完整流程演示
3. 生成统计报表
4. 查询设备信息
5. 启动定时任务调度器
6. 查看设备统计信息
0. 退出系统
```

### 4. 完整流程演示

选择菜单选项"2"即可运行从设备申请到报废的完整流程演示。

## 配置说明

编辑 `config.py` 可配置以下参数：

```python
BUDGET_APPROVAL_THRESHOLD = 500000  # 预算审批阈值（超50万需总监审批）
DEPRECIATION_MONTHS = 36            # 折旧月数（3年）
WARRANTY_MONTHS_DEFAULT = 12        # 默认保修期
SCHEDULED_TASK_TIME = '02:00'       # 日报生成时间
WECHAT_WEBHOOK_URL = ''             # 企业微信群机器人Webhook地址
```

## 高并发支持

- 数据库连接池配置：`pool_size=100, max_overflow=200`
- 支持批量数据处理：`chunk_list` 分批处理
- 事务管理完善：确保数据一致性
- 索引优化：关键查询字段建立索引
